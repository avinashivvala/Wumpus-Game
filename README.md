# Wumpus-Game

Implementation: –Reinforcement Learning based system (reward)

In the beginning of the game, it runs a simulation and records the rewards based on the percept found in the cloned world. Later the player tries to find the gold based on the rewards recorded.

Simulation: 
When the player has four possible moves available then player chose one of the moves. Then the new state may also have some possible moves. Likewise the player moves around the game until it finds the gold. Based on the precepts found in the game, rewards are awarded.
